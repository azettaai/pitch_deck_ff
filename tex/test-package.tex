\section{Simple Component Diagram}

Here's a simple diagram using the package:

\begin{figure}[ht]
\centering
\begin{tikzpicture}[node distance=2cm]

% Define nodes using package styles
\node (client) [cosma client] {Mobile App};
\node (api) [cosma component, below=of client] {API Gateway};
\node (db) [cosma storage, below=of api] {PostgreSQL};

% Draw connections
\draw [cosma data arrow] (client) -- node[right] {HTTP} (api);
\draw [cosma arrow] (api) -- node[right] {Query} (db);

\end{tikzpicture}
\caption{Simple Three-Tier Architecture}
\end{figure}
