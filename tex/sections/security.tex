\section{Security \& Compliance}

\subsection{Data Protection}

\begin{itemize}[noitemsep]
    \item \textbf{Encryption}: All data encrypted in transit (TLS 1.3) and at rest (AES-256)
    \item \textbf{Isolation}: Namespace-level separation, dedicated compute for premium
    \item \textbf{Privacy}: Customer data never used for model training
    \item \textbf{Logging}: Anonymized logs, configurable retention
\end{itemize}

\subsection{Compliance}

Cosma meets enterprise compliance requirements:

\begin{itemize}[noitemsep]
    \item SOC 2 Type II (in progress)
    \item GDPR compliant
    \item HIPAA-ready (enterprise tier)
    \item ISO 27001 certified data centers
\end{itemize}
